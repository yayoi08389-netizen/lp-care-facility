# lp-care-facility
未経験からの学習として制作した、特別養護老人ホームの紹介LPです。

## 概要
〜どんなLPか（誰向け・何を意識したか）〜

## 工夫した点
・レスポンシブ対応  

　画面幅によって画像が不自然に見切れないよう、background-position を調整しました。
  
・ハンバーガーメニュー（JavaScript） 

　クラスの切り替えで状態を管理し、JavaScriptは「状態制御」、CSSは「見た目制御」という役割分担を意識しました。
  
・アクセシビリティ

　文字サイズやHTMLタグの選択を工夫し、高齢者施設を探すご家族にも読みやすい構成を心がけました。
